// NBA Trivia CLI Game
A command-line trivia game that test your knowledge of NBA legends built with Javasript.

// Features

Interactive command-line interface
Real-time feedback on answers
Score tracking throughout the game
Performance evaluation based on the final score
Play again functionality
Input validation
Timed questions

// Technoligies Used

Node.js- JavaScript runtime
Readline- Built-in Node.js module for command-line input
Modular JavaScript- Organized code with seperate concerns

// Installation

Clone or download this project
Navigate to the project directory

cd trivia-cli

npm install

npm start or node index.js
